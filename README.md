#  clipboardEx plugin

Author: ゆんゆん探偵


## これはなに？

吉里吉里のクリップボード機能を拡張して、以下の機能追加をするプラグインです。

- クリップボードを介してBITMAP画像をやりとり出来るようにする
- クリップボードを介してTJS式をやりとり出来るようにする
- ウィンドウに、クリップボードの更新を検知する機能を追加する
- 吉里吉里および吉里吉里Zの、Clipboard.asTextを実行した際にテキストをコピーできないことがある不具合を修正する。

## 使い方

manual.tjs 参照

## ライセンス

このプラグインのライセンスは吉里吉里Zに準拠します。
